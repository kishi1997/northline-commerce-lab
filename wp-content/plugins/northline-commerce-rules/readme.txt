== Description ==

This plugin is not a legal-compliance or identity-verification product. It demonstrates secure WordPress input handling, WooCommerce CRUD APIs, HPOS compatibility, Classic Checkout validation, Checkout Block validation, and reusable WP-CLI fixtures.
